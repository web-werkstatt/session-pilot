{# Erklaer-Modal fuer den Workflow-Ring (Phase 7, 2026-04-14) #}

Workflow-Ring — wie ist das gemeint?

Der Ring zeigt den Lebenszyklus einer Arbeits-Aufgabe (Marker) in 5 Phasen. Er leuchtet immer an der Phase, die als naechstes Aufmerksamkeit braucht.

Begriffe

  • Projekt — ein Verzeichnis unter /mnt/projects/
  • Sprint-Plan — ein groesseres Vorhaben (z.B. "Dispatch Sprint 2a"); lebt in project_plans + sprints/*.md
  • Marker — eine konkrete, abgrenzbare Aufgabe innerhalb eines Plans. Eine Card = ein Marker. Liegt in handoff.md + DB-Tabelle markers
  • Session — eine Claude-Code-Session, die an einem Marker gearbeitet hat

Die 5 Phasen

  1. Gate Ready — Prompt & Checks liegen vor, Marker kann gestartet werden. (rot wenn Prompt oder Checks fehlen)
  2. Aktiv — Marker ist auf in_progress, aber noch keine Session dran. Naechster Schritt: Session starten.
  3. Execution — Tool/Agent arbeitet gerade (last_session gesetzt). Warten/beobachten.
  4. Write Back — Marker ist done oder blocked, Session-Ergebnis muss in die .md. Handoff oeffnen.
  5. Rating — done-Marker hat noch keinen execution_score. Score (0-5) + Kommentar vergeben.

Welcher Knoten leuchtet?

Priorisiert — der erste Treffer gewinnt:

  1. Gibt es irgendeinen done-Marker ohne Rating? → Rating
  2. Aktueller Marker ist in_progress und hat Session-Referenz? → Execution
  3. Aktueller Marker ist in_progress (noch keine Session)? → Aktiv
  4. Sonst: done/blocked-Marker vorhanden? → Write Back
  5. Sonst: naechster offener Marker? → Gate Ready

Farben

  • Haken — Phase fuer aktuellen Fokus erledigt
  • Blau — aktuelle Phase
  • Gelb/Orange — Handlungsbedarf (Write-Back, Rating)
  • Rot — Gate blockiert (Prompt/Checks fehlen)
  • Grau — noch nicht dran

Was in der Mitte steht

Der Notice-Block rechts vom Ring zeigt den Titel und Plan des aktuell fokussierten Markers. CURRENT = der Marker auf den sich der Ring-Status bezieht, NEXT = der naechste vorgesehene Marker.

Kurz

Das Bild ist eine Ampel fuer den aktuellen Marker-Fokus — keine Board-Uebersicht. Es sagt dir welche naechste Aktion fuer den im Fokus stehenden Marker ansteht.
